Calculator Demo

This is a demo of a very basic calculator made alongside my Just IT SWE Bootcamp, this is the version as of Week 2 Day 3

New Features (Built upon Week 2 Day 3):

  • Sucessfully added a form of button animation using basic css and html.
  • Sucessfully added a history to track previous equations!
  • Sucessfully added another equalsClickEval() function that uses eval, but I am still aiming to complete my own method to do this which can be used via equalsClickCustom()

Known Issues

  • Equations MUST be written as x*(y+z), eval() does not understand x(y+z) to be the same thing!
  • Custom equals can only handle 1 symbol out of: + - * /
  • Custom equals CAN NOT handle brackets as of this version.
